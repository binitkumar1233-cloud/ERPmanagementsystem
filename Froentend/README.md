# School & College ERP — Frontend

React + Vite frontend for the School and College ERP Management System.

## Setup

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server on http://localhost:3000 |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |

## Stack

- React 18
- React Router v6
- Tailwind CSS
- Lucide React (icons)
- Firebase
- Vite
